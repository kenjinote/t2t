## 概要
t2t は、C++ / Win32 API / Direct2D を用いてフルスクラッチで開発された、Windows向けの超軽量なテキストベースWebブラウザです。
HTMLからテキスト、リンク、基本的なフォーム要素のみを抽出し、DirectWriteによる高速かつ美麗なテキストレンダリングで画面に描画します。マウス操作だけでなく、Vimライクなキーボード操作にも対応しており、余計な装飾を省いたストイックな情報収集に最適化されています。

## 主な特徴
* **ネイティブ実装による高速描画:** Direct2D / DirectWrite を用いたハードウェアアクセラレーション描画。
* **キーボード・ドリブン:** `j` / `k` キーによるスクロール、`Tab` による要素移動、`/` によるページ内検索など、キーボードから手を離さずにブラウジングが可能。
* **レトロなテキストUI:** 独自のドットフォントをバイナリに内蔵し、コンソールライクでノスタルジックな表示画面を実現。
* **フォーム対応:** テキスト入力、ラジオボタン、チェックボックスに対応。DuckDuckGoなどの軽量エンドポイントを利用したWeb検索が標準で可能（`about:home`）。

## セキュリティについての考え方
本ブラウザは、堅牢で安全なテキスト閲覧環境を提供するため、以下のセキュリティ方針に基づいて設計されています。

* **プロトコルの厳格な制限:**
  ネットワーク通信には `libcurl` を使用していますが、接続可能なプロトコルを `HTTP` および `HTTPS` に限定しています。万が一、悪意のあるローカルパス（`file://` 等）や特殊なスキームがURLとして渡された場合でも、通信ライブラリの層で完全にブロックされます。
* **任意のコード実行の防止:**
  取得したデータはすべて「単なるテキストデータ」としてのみメモリ上で処理・描画されます。仮に実行ファイル（`.exe` など）のURLを開いたとしても、Windowsのプロセスとして起動・実行されることは構造上絶対にありません。
* **JavaScript非対応による安全性の担保:**
  JSエンジンを持たないプレーンなテキストパーサーを採用しているため、トラッキングスクリプトの動作や、クロスサイトスクリプティング（XSS）による脆弱性リスクが根本的に存在しません。

## 使用フォントについて
本アプリケーションのUIおよびWebページの描画には、レトロで美しいピクセルフォントである「ノスタルドット（M+）」をアプリケーションに内蔵（埋め込み）して使用しています。素晴らしいフォントの提供者に感謝いたします。

* **ノスタルドット（M+）**
* 配布元ページ: [https://free-fonts.jp/nosutaru-dot/](https://free-fonts.jp/nosutaru-dot/)
